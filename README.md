**HyFeChats** is an open-source Hytale plugin that adds private chat functionality for your server. It supports both
direct messages and group chats, allowing players to communicate discreetly and efficiently.

## Installation

0. Make sure you have installed the latest version of
   FeliUtils-Hytale: [CurseForge](https://curseforge.com/hytale/mods/feliutils) | [GitHub](https://github.com/Feli499/FeliUtils-Hytale).
1. Download the latest release of
   HyFeChats: [CurseForge](https://curseforge.com/hytale/mods/hyfechats) | [GitHub](https://github.com/Feli499/HyFeChats/releases/latest)
2. Place the `.jar` file in your server's `plugins` folder.
3. Restart your Hytale server.
